Color Picker

USING HCL INTERPOLATION

by Jared Wilber · October 2017

If you're anything like me, you probably spend a lot of time selecting the right color for your charts.
Choosing a color palette for a visualization can be surprisingly time consuming. Luckily, multiple websites exist to ameliorate this problem:

  • ColorBrewer2
  • HTML Color Picker
  • coolors
  • Colorpicker for data

Here's another tool to help with the process. Given two colors, 600 color-values will be interpolated between the two. Clicking on a color will decrease its opacity. Hovering over a color will yield its hexidecimal representation.
